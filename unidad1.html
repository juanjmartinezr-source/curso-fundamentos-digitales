1️⃣ Sistemas de Numeración y Conversiones

Los circuitos digitales trabajan con dos únicos estados (0 y 1), por lo que el sistema binario es la base de toda la electrónica digital. También se usan los sistemas octal y hexadecimal como formas compactas de representar valores binarios.

Sistemas más usados:

  • Binario (base 2): dígitos 0 y 1. Ej: 1011₂
  • Decimal (base 10): dígitos 0-9. Ej: 11₁₀
  • Octal (base 8): dígitos 0-7. Ej: 13₈
  • Hexadecimal (base 16): dígitos 0-9 y A-F. Ej: B₁₆

Conversión binario → decimal

1011₂ = (1×2³) + (0×2²) + (1×2¹) + (1×2⁰) = 8 + 0 + 2 + 1 = 11₁₀

3️⃣ Álgebra de Boole: Teoremas y Simplificación

El álgebra de Boole permite simplificar expresiones lógicas antes de construir el circuito, usando menos compuertas y reduciendo costos.

Teoremas y leyes principales

IdentidadA + 0 = A   |   A · 1 = A
NuloA + 1 = 1   |   A · 0 = 0
IdempotenciaA + A = A   |   A · A = A
ComplementoA + A' = 1   |   A · A' = 0
Doble negación(A')' = A
Ley de Morgan(A·B)' = A' + B'   |   (A+B)' = A'·B'
DistributivaA·(B+C) = A·B + A·C
AbsorciónA + (A·B) = A

Ejemplo de simplificación

F = A·B + A·B' + A'·B
= A·(B+B') + A'·B   (distributiva)
= A·1 + A'·B   (complemento)
= A + A'·B   (identidad)
= A + B   (absorción extendida)

4️⃣ Mapas de Karnaugh (2, 3 y 4 variables)

El mapa de Karnaugh es un método gráfico para simplificar expresiones booleanas agrupando los unos (minterms) en bloques de 1, 2, 4 u 8 celdas adyacentes.

Ejemplo: Mapa K de 2 variables

B=0B=1
A=001
A=111

Agrupando las celdas en 1 se obtiene la expresión mínima: F = A + B

Reglas de agrupación

  • Los grupos deben tener tamaño potencia de 2 (1, 2, 4, 8...)
  • Cuanto más grande el grupo, más simple el término resultante
  • Las celdas se pueden compartir entre varios grupos
  • El mapa "envuelve" en los bordes (la última columna es adyacente a la primera)
